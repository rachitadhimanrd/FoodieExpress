🍔 Foodie Express - Online Food Ordering System
Foodie Express is a modern, full-stack web application designed to provide a seamless food ordering experience. From browsing top-rated restaurants to real-time order tracking, this platform combines a premium user interface with a robust backend architecture.

🌐 Live Demo - https://foodieexpress-g183.onrender.com/

✨ Key Features
- Dynamic Restaurant Discovery: Browse restaurants by category, cuisine, or location with real-time filtering.
- Interactive Menu System: High-quality food imagery with detailed item descriptions and pricing.
- Smart Shopping Cart: Seamlessly add/remove items and manage quantities with automatic total calculation.
- User Authentication: Secure signup and login system with hashed password protection.
- Admin Dashboard: Full-featured management console to track orders, users, and restaurant statistics.
- Responsive Design: Fully optimized for mobile, tablet, and desktop viewing.

🛠️ Technology Stack
- Frontend: HTML5, CSS3 (Glassmorphism design), JavaScript (ES6+).
- Backend: Python 3.10+, Flask Framework.
- Database:
- SQLite (used for live deployment on Render).
- MySQL (supported for local development and large-scale scaling).
- ORM: SQLAlchemy (for database-agnostic management).
- Authentication: Flask-Login.
- Hosting: Render.com.
